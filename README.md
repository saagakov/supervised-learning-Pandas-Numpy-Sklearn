# supervised-learning-Pandas-Numpy-Sklearn

Practice project on supervised learning using Python and variety of its' libraries. 

Libraries used:
- Pandas
- Numpy
- Sklearn
- Seaborn
- matplotlib.pypython

Consists of:
- Standardization
- K-Nearest Neighbors (k-NN) classification
- Leave-one-out cross validation for classification accuracy
- Visualizing results
- Ridge regression
- K-Fold cross-validation for hyperparameter tuning
